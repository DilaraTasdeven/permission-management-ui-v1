RBAC AUTHORIZATION PANEL

Biga Bilişim bünyesinde geliştirilmiş, firma bazlı kullanıcı, rol ve yetki yönetimi sağlayan modern bir RBAC yönetim paneli.

Demo

Canlı önizleme:
https://github.com/DilaraTasdeven/permission-management-ui-v1

Özellikler
-Multi-tenant (firma bazlı yapı)
-Kullanıcı, rol ve yetki yönetimi
-Seviye bazlı izin matrisi
-Onay mekanizması
-Kritik log takibi
-Denetim (audit) kayıtları
-Responsive tasarım
